Die Printshop Integration Box (PIB) ist ein frei verfügbares Tool, um einen Druckbetrieb an PerfectPatterns sPrint One anzubinden. PIB sorgt ebenso für das Konvertieren und Hochladen von Druckbestellungen wie für das Herunterladen und Ausschießen produktionsfertiger Druckaufträge.

The Printshop Integration Box (PIB) is a freely available tool for connecting a print shop to PerfectPatterns sPrint One. PIB handles print order conversion and uploading, as well as download and imposition of production-ready print jobs.
